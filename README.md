# יומן פרויקטים — לבנט

אפליקציית דפדפן לניהול טיים ליין רישוי של פרויקטי נדל"ן.

## שימוש

פותחים את `index.html` בדפדפן. אין צורך בשרת או התקנה.

## יכולות

- ריבוי פרויקטים — כל פרויקט חדש מקבל טיים ליין רישוי מלא של 15 שלבי ברירת מחדל (מתכנון ראשוני ועד אגרות והשבחה)
- תאריך התחלה וסיום לכל שלב, עם חישוב משך אוטומטי
- הוספת עיכוב בין כל שני שלבים — שדה חופשי לגורם המעכב (יזם, עירייה, יועץ...) וספירת משך העיכוב; עיכוב ללא תאריך סיום נספר עד היום
- הוספת שלבים מותאמים אישית בכל נקודה בציר, מחיקת שלבים ועריכת שמות
- החלפת קטגוריית שלב (תכנון / עירייה / אבן דרך) בלחיצה על העיגול
- סיכום: עבודה נטו, סך עיכובים, ומשך מקצה לקצה
- ייצוא כל הפרויקטים לקובץ אקסל (גיליון לכל פרויקט, כולל עיכובים וסיכומים)
- שמירה אוטומטית ב-localStorage של הדפדפן + גיבוי ושחזור JSON

## הערה

הנתונים נשמרים מקומית בדפדפן. לגיבוי או העברה בין מחשבים — כפתור "ייצוא".
